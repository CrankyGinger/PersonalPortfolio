Chronoshift Expeditions

The first website I ever programmed from start to finish is the Chronoshift Expeditions website, a website for a time travel agency looking to give its customers more information about the company and possible destinations. The website contained information about a lot of different topics, such as company history and a brief description of all the different destinations that customers could travel to.

Per the requirements, we were instructed to use HTML and CSS only, which was entirely written in Visual Studio code. I wanted to make a website that featured plenty of images for visual aid to the user, and a good amount of text to describe the images and what they mean. There were also strict guidelines about the use of images, as they had to be free-use. Additionally, the HTML code had to be validated, which meant a lot of editing was in order.

When making the website, there were plenty of things that became a challenge, especially when making the web pages look presentable. The wireframe prototype that I created in Figma and the finished product are very similar in design and layout, but there were some revisions made. There were certain aspects of the wireframe that I could not figure out how to program using CSS - so I ended up making changes to accommodate this fact.

When making the website, I would become very frustrated, admittedly because of CSS. I was greatly enjoying the process, but it was very difficult for me to achieve certain things. When making the site, the use of the 'persist' and 'investigate' CART habits of mind were incredibly helpful. When something wasn't working or even breaking something else, I would investigate as to why, and persist through the difficulties. Unfortunately though, because of my limited experience it would take a long time to figure out what the issue was, but I persisted and finished the project.
